Just registered?  You need to do a few things.

- Register an account with [Code Forces](https://codeforces.com).  We recommend (but do not require) making a throwaway account for this class, especially if you happen to have taken the course before.  Note that your username and standings will be visible to everyone in the class.
- Tell us your username at this [Google Form](https://docs.google.com/forms/d/e/1FAIpQLSeU9nqksGZKlXaTaoTL1kSAZ4wXZ8KvocpAmXUr8haDyy9wTg/viewform?usp=sf_link)
- Set up [[Your GitHub Repository]] so you can see your grades.
- You might also want to read the [[Syllabus]].
